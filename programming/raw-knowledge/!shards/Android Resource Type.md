W Androidzie są różne typy resourców. Definiuje się je w różnych 
- animacje
	- res/animator/
	- res/anim/
- res/drawable
- res/layout/
- res/menu/
- res/color/
- string resources
	- res/values/
		- string/
- styles
	- -res/values
- res/font/

https://developer.android.com/guide/topics/resources/available-resources
